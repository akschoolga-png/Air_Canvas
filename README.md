# Air_Canvas

HOW TO USE
  - Run script and allow camera access
  - Use index finger to select color
  - Use index finger to draw
  - press 'q' to quit

REQUIREMENTS
  - Mediapipe (tasks API)
  - Open CV / cv2
  - NumPy

REFERENCES/CITATIONS
  - geeksforgeeks "Create Air Canvas Using Python-OpenCV"
    - for drawing and color selection
    - original version used color tracker
